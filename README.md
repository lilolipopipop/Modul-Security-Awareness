# Modul-Security-Awareness
Modul **Security Awareness** berbasis HTML interaktif yang dirancang untuk meningkatkan kesadaran pengguna terhadap ancaman **phishing** dan teknik **social engineering**.

Modul ini menyajikan materi pembelajaran secara bertahap melalui simulasi, studi kasus, dan panduan mitigasi sehingga pengguna dapat memahami cara mengenali serta merespons upaya phishing dengan lebih baik.

## Fitur
- 📖 Modul pembelajaran interaktif berbasis slide
- 🎯 Simulasi analisis email phishing
- 🔍 Panduan deteksi menggunakan metode **SCAN**
- 🛡️ Langkah mitigasi apabila akun terindikasi terkompromi
- 🌙 Dukungan Light & Dark Mode
- 📱 Tampilan responsif

## Tujuan
Modul ini dikembangkan sebagai media edukasi dalam program **Security Awareness** untuk membantu pengguna mengenali ancaman phishing, meningkatkan kewaspadaan, serta mendorong perilaku yang lebih aman dalam menggunakan layanan digital.
